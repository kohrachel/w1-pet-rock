# 🪨 Pet the Rock

This is my project for Week 1. It’s a rock. 🪨 A very innocent rock. No secret messages. No hidden agendas. Just a rock you should pet. 👇🐾 

Inspired by [Neal.fun](https://neal.fun/), built with React, TypeScript, and Vite ⚡️.

Total time taken: 1.5 hours + 2 hours debugging 

🛠️ Techstack
- TailwindCSS 💅
- Cursor Pro 🖱️
- Audio: [Pixabay](https://pixabay.com/) 🎵
- TTS: [TTS Tool](https://ttstool.com/) 🗣️

### 📦 Installation

1.  Clone the repository:
    ```bash
    git clone git@github.com:kohrachel/w1-pet-rock.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd w1-pet-rock
    ```
3.  Install the dependencies:
    ```bash
    npm install
    ```

Now go pet that rock. 🪨
